## Lessons

- The first screenshot was only sent to canvas and then deleted, so the supervisor could not prove the artifact persisted.
- AgInTiFlow was updated to persist canvas file artifacts into session storage.
- Android workflow guidance now asks for durable screenshot paths and final git status.
