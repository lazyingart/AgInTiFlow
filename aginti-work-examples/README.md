# AgInTiFlow Work Examples

This directory records supervised AgInTiFlow homework runs. It is not a generic demo gallery; each entry should include evidence that AgInTiFlow actually completed the task.

## Rule

Do not record a task as complete only because the agent said it was complete. Each example needs durable proof such as:

- source files or commits,
- build/test logs,
- screenshots or PDFs,
- session id and tmux session name,
- artifact paths that still exist,
- notes about flaws found and AgInTiFlow upgrades made.
