# AgInTiFlow + AAPS Scientific And Industrial Positioning

Date: 2026-05-06

This reference records the preferred product language for positioning AgInTiFlow and AAPS together. It is intentionally duplicated in both repositories so either project can be described without relying on the other checkout.

## Core Pairing

AgInTiFlow and AAPS should be described as two layers of the same broader system:

| Product | Role |
| --- | --- |
| AgInTiFlow | Project-aware agent workspace and runtime. |
| AAPS | Project-aware / project-native workflow programming language and visual studio. |

Short relationship:

> AAPS describes the workflow. AgInTiFlow runs, supervises, and verifies the work.

## Preferred AgInTiFlow Description

### Title

AgInTiFlow

### Subtitle

Project-aware agents for scientific and industrial work.

### Hero Text

AgInTiFlow is a project-aware agent workspace for hybrid wet-dry R&D, hardware-aware intelligence, and industrial automation. From wet-lab planning to dry-lab analysis, from microscopy images to production scripts, AgInTiFlow helps agents work inside real project context through SCS supervision, AAPS workflows, guarded tool use, durable sessions, and API, Web, or CLI entry points.

### More Specific Version

AgInTiFlow is a project-aware Web and CLI agent workspace for chemistry, biology, imaging, software, hardware-aware intelligence, and industrial automation. It keeps agents grounded in real project context: files, protocols, data, tools, commands, artifacts, logs, and previous sessions. With visible tool calls, durable session storage, guarded local execution, model routing, scouts, SCS supervision, and AAPS workflow support, AgInTiFlow helps turn open-ended scientific and industrial tasks into inspectable, resumable, and verifiable work.

### Chinese Version

AgInTiFlow 是一个面向混合干湿实验、科学研发与工业应用的 project-aware agent 工作空间。

它不是单纯的聊天机器人，而是让 agent 真正工作在项目之中：理解文件、协议、数据、图像、脚本、工具、命令、日志、artifact 和历史 session。AgInTiFlow 通过 Web 与 CLI 提供可见的工具调用、可恢复的任务会话、受保护的本地执行、模型路由、scout 分析、SCS 监督机制，以及 AAPS 工作流支持，使化学、生物、成像、软件和工业任务能够被规划、执行、验证、恢复和复现。

## Preferred AAPS Description

### Title

AAPS

### Subtitle

Project-aware programming language for science, software, and industry.

Alternative subtitle:

Project-native workflow programming for agents.

### Hero Text

AAPS is a prompt-native language and visual studio for turning prompts into structured pipelines to connect wet and dry experiments, hardware and software, with agents, tasks, typed inputs, declared outputs, executable actions, validation gates, recovery steps, and durable artifacts.

AAPS is a project-aware scripting language and visual studio for building structured agent pipelines. It turns prompts, protocols, scripts, tools, data, and validation gates into workflows with agents, tasks, typed inputs, declared outputs, executable actions, recovery steps, and durable artifacts for science, software, imaging, writing, and industrial workflows.

它把 prompt、实验协议、脚本、工具、数据和验证规则组织成结构化 agent pipeline，通过 agents、tasks、typed inputs、declared outputs、executable actions、validation gates、recovery steps 和 durable artifacts，让自主任务能够被规划、执行、验证、恢复和复现。

Recommended English:

- Project-Native Programming

- Project-Native Workflow Programming
- Project-Oriented Agentic Programming

Recommended pairing:

- AAPS: Project-native workflow programming.

### Hardware Intelligence

Prefer one of these unless using "hardware intelligence" as a deliberate brand phrase:

- hardware-aware intelligence
- hardware-integrated intelligence
- AI for intelligent hardware and industrial automation
