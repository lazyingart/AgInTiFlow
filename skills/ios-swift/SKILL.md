# iOS Swift Development

Inspect before editing: `Package.swift`, `.xcodeproj`, `.xcworkspace`, schemes, `Info.plist`, assets, signing settings, `xcodebuild -version`, and available simulators.

Prefer Swift Package Manager or project-local Xcode build commands. Verify with `swift test`, `xcodebuild build/test`, simulator launch, or a clear blocker when Xcode/signing/simulator access is unavailable. Save screenshots under durable workspace paths when a simulator is available.
